== Description ==

The best CSS3 buttons Generator Plugin!

**Now with TinyMCE shortcode Generator Button, so no need to remember the Shortcode and its parameters, Just Paste the Parameters in the generator form and hit the 'Generate Shortcode' Button and BOOM!!! there you go with your shortcode.**

*Use it THEN Rate it. And Share the Love*

Features

* @Font-Face icons
* Custom Font file for all social icons
* em sizing based on button font-size
* Support for about **73** different Services
* No raster images (sweet)
* Editable button text
* Editable size
* One version for all resolutions
* 100% vector icon with @font-face
* Cross-browser support
* Works splendidly on any browser supporting @font-face
* Gracefully degrades (IE6, IE7)
* Button & Icon styles
* Also includes generic icon-less primary and secondary buttons. And Call, Email, Calender, CArt, Print buttons with Icons.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Read the Documentation for using the plugin.
4. Use the shortcode for using the buttons.
